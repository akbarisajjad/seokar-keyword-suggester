افزونه‌ای هوشمند برای پیشنهاد کلمات کلیدی، عنوان و توضیحات متا با استفاده از OpenAI.  

== Description ==  
**🚀 هوش مصنوعی OpenAI برای سئو بهتر وردپرس!**  
افزونه **سئوکار پیشنهاد کلمه کلیدی (Seokar Keyword Suggester)** به شما کمک می‌کند **کلمات کلیدی، عناوین سئو و توضیحات متا** را به‌صورت **هوشمندانه و خودکار** دریافت کنید.  

**📌 ویژگی‌های کلیدی:**  
✔ پیشنهاد **کلمات کلیدی مرتبط** برای نوشته‌ها، محصولات و صفحات وردپرس  
✔ پیشنهاد **عنوان سئو (SEO Title) و توضیحات متا (Meta Description)**  
✔ پشتیبانی از **چندین زبان** (فارسی، انگلیسی، عربی، آلمانی و …)  
✔ قابلیت **ذخیره خودکار پیشنهادات در متادیتای وردپرس**  
✔ **رابط کاربری زیبا** و طراحی سبک برای افزایش سرعت سایت  
✔ **سازگاری کامل با ووکامرس، WPML و Polylang**  

== Installation ==  
۱. افزونه را از [GitHub](https://github.com/sajjadakbari/seokar-keyword-suggester) یا مخزن وردپرس دانلود کنید.  
۲. فایل `seokar-keyword-suggester.zip` را در `/wp-content/plugins/` آپلود کنید.  
۳. از پیشخوان وردپرس، افزونه **سئوکار پیشنهاد کلمه کلیدی** را فعال کنید.  
۴. به **تنظیمات > سئوکار پیشنهاد کلمه کلیدی** بروید و **API Key** خود را وارد کنید.  

== Frequently Asked Questions ==  
**۱. API Key چیست و چگونه دریافت کنم؟**  
✅ به [OpenAI](https://platform.openai.com/signup/) مراجعه کنید.  
✅ ثبت‌نام کرده و از بخش [API Keys](https://platform.openai.com/account/api-keys) یک کلید جدید ایجاد کنید.  
✅ API Key را در تنظیمات افزونه وارد کنید.  

**۲. آیا افزونه باعث کاهش سرعت سایت می‌شود؟**  
❌ خیر، افزونه سبک طراحی شده و فقط در صورت نیاز درخواست API ارسال می‌شود.  

**۳. آیا افزونه رایگان است؟**  
✅ بله، اما برای استفاده از OpenAI باید **API Key تهیه کنید** که ممکن است هزینه داشته باشد.
